This project builds a hybrid-local AI IT Service Desk Copilot for managed IT service scenarios. The system classifies support tickets, predicts urgency, retrieves similar historical incidents, and generates troubleshooting recommendations using a local RAG pipeline.

All computation is performed locally using Python, scikit-learn, SentenceTransformers, FAISS, Ollama, FastAPI, and Streamlit. Azure Blob Storage is used only for enterprise-style artifact backup and project storage. This design demonstrates both hands-on AI implementation ability and understanding of Microsoft/Azure enterprise workflows.
